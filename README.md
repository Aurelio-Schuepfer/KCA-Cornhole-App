## Getting Started

### Prerequisites

- Node.js (v14+)  
- Angular CLI (`npm install -g @angular/cli`)

### Install & Run

git clone https://github.com/Aurelio-Schuepfer/KCA-Cornhole.git
cd KCA-Cornhole
npm install
ng serve
Open http://localhost:4200
